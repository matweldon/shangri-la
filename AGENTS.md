# Python

- Use uv `add`, `sync` etc
- numpy docstrings and type hints for everything
- Typehints: | syntax not Union
- unit tests for everything
- Functional core; imperative shell

# Javascript/Typescript

- Minimal dependencies
- Typescript strict mode
- Single quotes. Semicolons
- Use functional patterns where possible
- No jquery syntactical sugar
- No component frameworks

# Architecture

- Local first
- Static front-end apps with servers for discrete services, or use BaaS
- Use URL navigation (including # sections) before considering SPA-style navigation

# Design

- Simple
- Always mobile-friendly unless explicitly stated otherwise (if mobile-friendly would be impractical, ask for advice)
